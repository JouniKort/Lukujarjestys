# Lukujarjestys
Ohjelmalla voi tehdä lukujärjestyksiä Lappeenrannan teknillisen yliopiston kursseista. 
Kurssien tiedot on ladattu osoitteesta https://uni.lut.fi/lukujarjestykset1 ja talletettu Access-tietokantaan.
Jotta ohjelma saisi yhteyden Accessiin, voi joutua lataamaan paketin osoitteesta http://www.microsoft.com/en-us/download/details.aspx?id=5793
Ohjelma käyttää .NET Framework versiota 4.5.2 jonka voi ladata osoitteesta https://www.microsoft.com/fi-fi/download/details.aspx?id=42642
